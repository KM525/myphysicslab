This simulation uses RigidBodySim instead of the usual ContactSim. It handles forces like springs, gravity, damping, and thrust forces but does not calculate contact forces or collision impulses. Therefore objects pass thru each other. Activate thrust forces by clicking in the canvas and then use arrow keys or the letter keys "jkli" or "sdfe".
